Quick Setup Wizard V2 Debug

Test Instructions:

  1. Open the wizard URL: Open Wizard
  2. Open browser console (F12)
  3. Check for any JavaScript errors
  4. Run this in console to check Livewire data:
// Check if Livewire is loaded
console.log('Livewire loaded:', typeof Livewire !== 'undefined');

// Get the component
let component = Livewire.find(document.querySelector('[wire\\:id]').getAttribute('wire:id'));

// Check the data
console.log('Component data:', component.data);
console.log('Services:', component.data.data?.services);
console.log('Staff:', component.data.data?.staff);

// Check form state
console.log('Edit mode:', component.editMode);
console.log('Company ID:', component.data.data?.company_id);

Common Issues:

  • Empty form: Data is loaded but not displayed - likely a form state issue
  • No services/staff: Data loading failed - check Laravel logs
  • JavaScript errors: Alpine.js or Livewire issue - check console

Manual Data Check:

Run this PHP script to verify data exists:

php test-wizard-data-simple.php 1
