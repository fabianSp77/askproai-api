Quick Start ​

Get AskPro API Gateway up and running in minutes.

Prerequisites ​

  • PHP 8.2+
  • MySQL 8.0+
  • Redis
  • Node.js 18+
  • Composer

Installation ​

1. Clone the Repository ​

bash
git clone https://github.com/fabianSp77/askproai-api.git
cd askproai-api
1
2

2. Install Dependencies ​

bash
composer install
npm install
1
2

3. Environment Setup ​

bash
cp .env.example .env
php artisan key:generate
1
2

4. Configure Database ​

Edit .env with your database credentials:

env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=askproai
DB_USERNAME=your_username
DB_PASSWORD=your_password
1
2
3
4
5
6

5. Run Migrations ​

bash
php artisan migrate
php artisan db:seed
1
2

6. Build Assets ​

bash
npm run build
1

7. Start the Server ​

bash
php artisan serve
1

Visit http://localhost:8000/admin to access the admin panel.

Default Credentials ​

EmailPassword
admin@askproai.de(set in seeder)
