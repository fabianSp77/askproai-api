Authentifizierung

Alle API-Anfragen erfordern einen gültigen Authentication-Header.

// Beispiel: Authorization-Header verwenden
const headers = {
  'Authorization': 'Bearer YOUR_API_KEY',
  'Content-Type': 'application/json'
};

fetch('https://api.askproai.de/v3/availability', {
  method: 'GET',
  headers: headers
});
