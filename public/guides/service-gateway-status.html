Konfiguration

config/gateway.php

return [ 'mode_enabled' => env('GATEWAY_MODE_ENABLED', false), 'default_mode' => env('GATEWAY_DEFAULT_MODE', 'appointment'), 'modes' => ['appointment', 'service_desk', 'hybrid'], 'hybrid' => [ 'fallback_mode' => env('GATEWAY_HYBRID_FALLBACK', 'appointment'), ], ];

Aktivierung

Schritt 1: In .env setzen:

GATEWAY_MODE_ENABLED=true

Schritt 2: Company Policy konfigurieren:

gateway_mode: 'service_desk'
