üéØ Project Overview

AskProAI is an AI-powered appointment booking platform that connects phone calls (Retell.ai) to calendar systems (Cal.com). Multi-tenant SaaS for German service businesses.

üöÄ NEW: Best Practices 2025 Implementation

‚úÖ What's Already Implemented

  1. Automatic MCP Server Discovery
  2. MCPAutoDiscoveryService - Finds best MCP server for any task
  3. UsesMCPServers trait - Add to any service for auto-MCP

  4. Command: php artisan mcp:discover "your task"

  5. Complete Data Flow Tracking

  6. DataFlowLogger - Tracks all external API calls
  7. Correlation IDs for end-to-end debugging
  8. Auto-generates sequence diagrams

  9. Command: php artisan dataflow:list

  10. System Self-Understanding

  11. SystemUnderstandingService - Analyzes existing code
  12. ImpactAnalyzer - Predicts deployment risks
  13. Prevents breaking changes automatically

  14. Command: php artisan analyze:impact

  15. Code Quality Automation

  16. Laravel Pint (formatting) - composer pint
  17. PHPStan Level 8 (analysis) - composer stan
  18. Git hooks for pre-commit checks
  19. Documentation auto-update reminders

üî• Common Issues & Quick Fixes

Top 3 Problems

  1. "No calls imported" bash php artisan horizon # Start queue # Then click "Anrufe abrufen" in admin

  2. "Database access denied" bash rm -f bootstrap/cache/config.php php artisan config:cache

  3. "Webhook signature invalid" bash # Check .env: RETELL_WEBHOOK_SECRET # Verify in Retell dashboard

üé® Development Workflow

Before ANY Task

# 1. Discover best MCP server
php artisan mcp:discover "describe your task"

# 2. Understand existing code
php artisan analyze:component ComponentName

# 3. Start data flow tracking
php artisan dataflow:start

During Development

  • Use $this->executeMCPTask() in services with UsesMCPServers trait
  • Track external calls with DataFlowLogger
  • Check impact with php artisan analyze:impact

Before Committing

# Quality checks (automatic via git hooks)
composer quality  # Runs all checks

# Documentation check
php artisan docs:check-updates
