# Passwort vergessen - was nun?

Kein Problem! Sie können Ihr Passwort jederzeit zurücksetzen.

## So setzen Sie Ihr Passwort zurück:

### 1. Gehen Sie zur Anmeldeseite
Öffnen Sie das Kundenportal und klicken Sie auf "Anmelden".

### 2. Klicken Sie auf "Passwort vergessen?"
Sie finden den Link unterhalb des Anmeldeformulars.

### 3. Geben Sie Ihre E-Mail-Adresse ein
Verwenden Sie die E-Mail-Adresse, mit der Sie registriert sind.

### 4. E-Mail abrufen
Sie erhalten innerhalb weniger Minuten eine E-Mail mit einem sicheren Link.

### 5. Neues Passwort festlegen
Klicken Sie auf den Link und wählen Sie ein neues, sicheres Passwort.

## Tipps für ein sicheres Passwort:
- Mindestens 8 Zeichen lang
- Kombination aus Buchstaben, Zahlen und Sonderzeichen
- Keine persönlichen Informationen (Geburtstag, Name)
- Für jedes Konto ein eigenes Passwort

## Häufige Probleme:

**Keine E-Mail erhalten?**
- Prüfen Sie Ihren Spam-Ordner
- Vergewissern Sie sich, dass die E-Mail-Adresse korrekt ist
- Warten Sie 5 Minuten und versuchen Sie es erneut

**Link funktioniert nicht?**
- Der Link ist nur 24 Stunden gültig
- Fordern Sie einen neuen Link an
- Kopieren Sie den kompletten Link aus der E-Mail
