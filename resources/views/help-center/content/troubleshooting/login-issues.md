# Login-Probleme beheben

Haben Sie Schwierigkeiten, sich in Ihrem Kundenportal anzumelden? Hier finden Sie Lösungen für die häufigsten Probleme.

## Häufige Login-Probleme

### 1. Passwort vergessen

**Problem:** Sie können sich nicht an Ihr Passwort erinnern.

**Lösung:**
1. Klicken Sie auf der Login-Seite auf **"Passwort vergessen?"**
2. Geben Sie Ihre E-Mail-Adresse ein
3. Klicken Sie auf **"Passwort zurücksetzen"**
4. Prüfen Sie Ihr E-Mail-Postfach (auch Spam-Ordner!)
5. Klicken Sie auf den Link in der E-Mail
6. Vergeben Sie ein neues Passwort

> **Tipp:** Wählen Sie ein sicheres Passwort mit mindestens 8 Zeichen, Groß- und Kleinbuchstaben sowie Zahlen.

### 2. E-Mail-Adresse nicht erkannt

**Problem:** Die Meldung "E-Mail-Adresse nicht gefunden" erscheint.

**Mögliche Ursachen:**
- Tippfehler in der E-Mail-Adresse
- Verwendung einer anderen E-Mail-Adresse bei der Registrierung
- Noch kein Konto erstellt

**Lösung:**
1. **Prüfen Sie die Schreibweise** Ihrer E-Mail-Adresse
2. **Versuchen Sie alternative E-Mail-Adressen**, die Sie verwenden
3. **Prüfen Sie alte E-Mails** nach Terminbestätigungen - dort steht Ihre registrierte E-Mail
4. **Erstellen Sie ein neues Konto**, falls Sie noch keines haben

### 3. Falsches Passwort

**Problem:** Die Meldung "Falsches Passwort" erscheint.

**Lösung:**
1. **Caps Lock prüfen** - Ist die Feststelltaste aktiviert?
2. **Leerzeichen vermeiden** - Vor oder nach dem Passwort
3. **Browser-Autofill prüfen** - Manchmal fügt der Browser alte Passwörter ein
4. **Passwort zurücksetzen** - Nutzen Sie "Passwort vergessen?"

### 4. Konto gesperrt

**Problem:** "Ihr Konto wurde aus Sicherheitsgründen gesperrt."

**Ursache:** Zu viele fehlgeschlagene Login-Versuche

**Lösung:**
1. **Warten Sie 30 Minuten** - Das Konto wird automatisch entsperrt
2. **Passwort zurücksetzen** - Für sofortigen Zugang
3. **Support kontaktieren** - Bei wiederholten Problemen

## Browser-spezifische Probleme

### Cookies und Cache

Manchmal verursachen gespeicherte Daten Probleme.

**So löschen Sie Cache und Cookies:**

**Chrome:**
1. Drei Punkte oben rechts → Einstellungen
2. Datenschutz und Sicherheit → Browserdaten löschen
3. Zeitraum: "Gesamte Zeit"
4. Haken bei "Cookies" und "Cache"
5. "Daten löschen" klicken

**Firefox:**
1. Drei Striche oben rechts → Einstellungen
2. Datenschutz & Sicherheit → Cookies und Website-Daten
3. "Daten entfernen" klicken

**Safari:**
1. Safari → Einstellungen → Datenschutz
2. "Website-Daten verwalten" → "Alle entfernen"

### JavaScript aktivieren

Unser Portal benötigt JavaScript.

**So aktivieren Sie JavaScript:**
- **Chrome:** Einstellungen → Datenschutz → Website-Einstellungen → JavaScript → Erlauben
- **Firefox:** about:config → javascript.enabled → true
- **Safari:** Einstellungen → Sicherheit → JavaScript aktivieren

## Sicherheitstipps

### Starke Passwörter verwenden
- Mindestens 8 Zeichen
- Groß- und Kleinbuchstaben
- Zahlen und Sonderzeichen
- Keine persönlichen Informationen

### Phishing vermeiden
- Wir fragen **niemals** per E-Mail nach Ihrem Passwort
- Prüfen Sie die URL: Nur über die offizielle Website einloggen
- Bei verdächtigen E-Mails: Nicht auf Links klicken!
